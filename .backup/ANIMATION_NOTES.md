## Important GitHub limitation

GitHub README files do not allow real JavaScript interaction for security. Also, SVGs used as images cannot reliably contain clickable internal elements.

So the best world-class profile interaction is:

- Animated visuals for attention
- Clickable badges and tables for navigation
- Live project links
- Portfolio links
- Demo buttons
- Screenshots/GIFs
- GitHub Actions-generated visuals like snake and 3D contributions
